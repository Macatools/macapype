    Pipelines¶

    Data preparation (macapype.pipelines.prepare):

    create_data_preparation_pipe(params[, name]) prepare: - av = checking if multiples T1 and T2 and if it is the case, coregister and average (~ flirt_average) - coregister T2 on T1 with FLIRT - crop using .cropbox file
    create_reorient_pipeline([name, new_dims]) By kepkee: fslswapdim image_bad x z -y image_good fslorient -deleteorient image_good.nii.gz; fslorient -setqformcode 1 image_good.nii.gz
