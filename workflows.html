Definition of workflows¶

SPM-based pipelines:¶

Using T1w and T2w images¶

SPM-based full pipeline, with the processing sequence:

  • Data preparation

  • T1xT2BiasFieldCorrection

  • IterREGBET

  • old_segment SPM based pipeline

Unsing only a T1w image¶

<macapype.pipelines.full_pipelines.create_full_spm_subpipes>`, with the processing sequence:

  • Data preparation

  • T1xT2BiasFieldCorrection but the T1w replaces the T2w

  • IterREGBET

  • ANTS non linear registration and FSL Apply XFM to tranform images to the template

  • old_segment SPM based pipeline

ANTS-based pipelines:¶

ANTS-based full pipeline, with the processing sequence :

  • Data preparation

    • Averaging multiple files from same nature

    • possibly reorientation pipeline

    • cropping (using T1xT2BET for cropping tool, or fslroi specifying cropbox)

    • denoising from Ants non-local mean

  • Brain extraction pipeline

    • debias pipeline (similar to T1xT2BiasFieldCorrection, but all steps are nipype nodes)

    • extract brain pipeline (using Atlax-Brex)

  • Brain segment from mask :

    • masked debias pipeline

    • register pipeline from template (NMT) to subject space

    • segmentation pipeline in subject space with Atropos

How to run workflows¶

To run the pipeline over a full BIDS dataset:

$ python workflows/segment_pnh_spm_based.py -data ~/Data_maca -out ./local_test

To specify with subjects/sessions you want to run the pipeline on:

$ python workflows/segment_pnh -data ~/Data_maca -out ./local_test -soft SPM -subjects Elouk -sess 01
$ python workflows/segment_pnh -data ~/Data_maca -out ./local_test -soft SPM -subjects Elouk

You can also specify a json file containing the parameters of your analysis

$ python workflows/segment_pnh.py -data ~/Data_maca -out ./local_test -soft SPM -params params.json

You can also include multi_params.json, for specifiying parameters specific parameters:

$ python workflows/segment_pnh.py -data ~/Data_maca -out ./local_test -soft SPM -params params.json -multi_params multi_params.json
