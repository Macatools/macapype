    Macapype¶

    Macapype is an open-source multi-modal brain data analysis kit which provides Python-based pipelines for advanced multi-thread processing of fMRI anat in PNH. Neuropycon is based on Nipype, a tool developed in fMRI field, which facilitates data analyses by wrapping many commonly-used neuro-imaging software into a common python framework.

    Installation¶

    See Installation for installation on your local system if you have adequate softwares (i.e. FSL, AFNI, Ants, SPM) running on your machine/clusters. Running subsequent workflows can be found here

    Examples¶

    An exemple of reorientation, starting from a sphinx position.

    The plot_segment_pnh_regis_T1xT2 workflow based on Regis’ full pipeline, with the processing sequence:

    • T1xT2Bet
    • T1xT2BiasFieldCorrection
    • IterREGBET
    • old_segment SPM based pipeline

    The plot_segment_pnh_kepkee workflow based on Kepkee’s full pipeline, with the processing sequence :

    • T1xT2Bet (used for the cropping tool only)
    • debias pipeline (~ to T1xT2BiasFieldCorrection, but all steps are nipype nodes)
    • denoise pipeline (from Ants non-local mean)
    • brain extraction pipeline using Atlax-Brex
    • full segment from mask (starting from the mask, the following steps are computed):
      • denoise pipeline
      • masked debias pipeline
      • register pipeline from template (NMT) to subject space
      • segmentation pipeline in subject space with Atropos

    The same full pipeline, used on marmouset data (link to example). The automated cropping did not work, hence manual cropping was used
