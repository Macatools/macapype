Macapype¶

Macapype is an open-source multi-modal brain data analysis kit which provides Python-based pipelines for advanced multi-thread processing of MRI anatomical data of PNH brain images. Neuropycon is based on Nipype, a tool developed in fMRI field, which facilitates data analyses by wrapping many commonly-used neuro-imaging software into a common python framework.

API, describing the different pipelines and nodes, can be found here

See Installation for installation on your local system if you have adequate softwares (i.e. FSL, AFNI, Ants, SPM) running on your machine/clusters.
