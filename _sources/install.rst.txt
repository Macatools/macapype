.. _install:

**************
How to install
**************

from github
############

Using git
---------

.. code:: bash

    $ git clone https://github.com/Macatools/macapype.git

    $ cd macapype

    $ python setup.py develop --user

Using pip
---------

.. code:: bash

    $ pip install git+https://github.com/Macatools/macapype
