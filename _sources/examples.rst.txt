Pipeline examples
__________________

Macaque:
---------

* :ref:`Segmentation of macaque test dataset <plot_segment_macaque_spm_based>` processed with SPM-based :class:`full pipeline <macapype.pipelines.full_pipelines.create_full_T1xT2_segment_pnh_subpipes>`.

Origin of the dataset: `ERC COVOPRIM <https://cordis.europa.eu/project/id/788240>`_ project.

Acknowledgements to Pascal Belin and Regis Trapeau.

|

* :ref:`Segmentation of macaque test dataset <plot_segment_macaque_ants_based>` processed with ANTS-based :class:`full pipeline <macapype.pipelines.full_pipelines.create_full_segment_pnh_subpipes>`.

Origin of the dataset: `ERC COVOPRIM <https://cordis.europa.eu/project/id/788240>`_ project.

Acknowledgements to Pascal Belin and Regis Trapeau.

|

* :ref:`Segmentation of macaque test dataset with reorientation <plot_segment_sphinx_macaque_ants_based>`, starting from a sphinx position, processed with ANTS-based :class:`full pipeline <macapype.pipelines.full_pipelines.create_full_segment_pnh_subpipes>`.

Acknowledgements to Frederic Chavane and Isabelle Racicot.

|

Marmoset:
----------

* :ref:`Segmentation of marmoset test dataset <plot_segment_marmo_ants_based>`, processed using same ANTS-based :class:`full pipeline <macapype.pipelines.full_pipelines.create_full_segment_pnh_subpipes>`.

The automated cropping did not work, hence manual cropping was used, cropbox has to be specified manually.

Origin of the dataset: `ANR Marmobrain <https://anr.fr/Projet-ANR-18-CE37-0017>`_ project.

Acknowledgements to Frederic Chavane, Thomas Brochier and Julien Sein.

|

Baboon:
--------

* :ref:`Segmentation of baboon test dataset <plot_segment_baboon_ants_based>`, processed using same ANTS-based :class:`full pipeline <macapype.pipelines.full_pipelines.create_full_segment_pnh_subpipes>`, with single_preparation pipeline for data_preparation.

Origin of the dataset: `ERC GestImage <https://cordis.europa.eu/project/id/716931/fr>`_ project.

Acknowledgements to Adrien Meguerditchian and Yannick Becker. ** dataset not yet available for download and test**
