Pipeline examples¶

SPM-based pipelines:¶

Segmentation of PNH test dataset processed with SPM-based full pipeline, with the processing sequence:

  • Data preparation
  • T1xT2BiasFieldCorrection
  • IterREGBET
  • old_segment SPM based pipeline

ANTS-based pipelines:¶

Segmentation of PNH test dataset processed with ANTS-based full pipeline, with the processing sequence :

  • Data preparation
    • Averaging multiple files from same nature
    • possibly reorientation pipeline
    • cropping (using T1xT2BET for cropping tool, or fslroi specifying cropbox)
    • denoising from Ants non-local mean
  • Brain extraction pipeline
    • debias pipeline (similar to T1xT2BiasFieldCorrection, but all steps are nipype nodes)
    • extract brain pipeline (using Atlax-Brex)
  • Brain segment from mask :
    • masked debias pipeline
    • register pipeline from template (NMT) to subject space
    • segmentation pipeline in subject space with Atropos

Segmentation of PNH test dataset with reorientation, starting from a sphinx position, processed with ANTS-based full pipeline.

Segmentation of marmoset test dataset, processed using same ANTS-based full pipeline. The automated cropping did not work, hence manual cropping was used, cropbox has to be specified manually.
