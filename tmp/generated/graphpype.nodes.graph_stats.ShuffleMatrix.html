    class graphpype.nodes.graph_stats.ShuffleMatrix(from_file=None, resource_monitor=None, ignore_exception=False, **inputs)[source]¶

    Compute randomisation of matrix, keeping the same distribution

    If seed = -1, no shuffling is done (keep original values)
